JavaScript Variables

    JavaScript variables are containers for data that you can reference throughout your code. Using variables enables you to store data, perform computations, and manipulate values dynamically.

    Common variable datatypes

  1. String - Textual data.
  2. Number - integer or a float.
  3. Boolean - true or false.
  4. Undefined - No value initialised.
  5. null - null value.
  6. BigInt - integer in the range of -2^63 to 2^63-1.
