JavaScript Events

    An event is like a notification that something has happened to an object. In HTML, there are different events that represent actions performed by the user or the browser, such as clicking a button or the page finishing loading. When we include JavaScript code in an HTML file, it can "listen" for these events and respond accordingly. JavaScript reacts to these events and allows us to execute specific actions or functions when those events occur. This process of reacting to events and executing associated code is called event handling.

    Within a event handler function, you perform DOM manipulation to modify HTML elements or their properties based on the event. For example, you can update the text of an element, change its style, add or remove elements, or perform other actions.

    Common Event Handlers

  1. onclick - Triggered when an element is clicked.
  2. onmouseover - Triggered when the mouse pointer moves over an element.
  3. onmouseout - Triggered when the mouse pointer moves out of an element.
  4. onkeydown - Triggered when a key is pressed down.
  5. onkeyup - Triggered when a key is released.
  6. onsubmit - Triggered when a form is submitted.
  7. onload - Triggered when a web page or an image finishes loading.
